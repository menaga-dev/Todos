# Todos Project Documentation

## Overview
This repository contains a simple Todos application to manage daily tasks.

## Features
- Add, edit, and delete tasks
- Mark tasks as completed
- Filter tasks based on status

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/menaga-dev/todos.git
   cd todos
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Start the application**:
   ```bash
   npm start
   ```
4. **Access the app** in your web browser at `http://localhost:3000`.
